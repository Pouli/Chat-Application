Chat-Application
================

A simple chat application using Node.js, Angular.js and Socket.io 

## Install
  - Node.js
  - npm
  - bower
  
## NPM
 ```sh
$ npm install
```

## Bower
 ```sh
$ bower install
```

## Node.js
 ```sh
$ node server
```

Finally go on http://localhost:8080/
